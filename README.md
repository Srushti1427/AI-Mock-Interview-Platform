# AI-Powered Mock Interview Platform

## Description
An advanced AI-powered mock interview platform designed to help users practice and improve their interview skills through real-time interaction, feedback, and performance analysis.

This application simulates real interview scenarios by generating dynamic questions, analyzing user responses, and providing actionable insights to enhance performance.

---

## Features
- AI-driven interview question generation and feedback
- Personalized interview sessions based on role and experience
- Speech-to-text evaluation using Whisper API
- Real-time feedback with performance insights
- Webcam-based proctoring using TensorFlow.js (BlazeFace)
- Interactive dashboard with recent interview history
- Chatbot support for interview preparation

---

## Tech Stack
- **Frontend:** Next.js, React, Tailwind CSS
- **Backend:** Next.js API Routes
- **Database:** PostgreSQL (Neon Serverless)
- **ORM:** Drizzle ORM
- **AI Integration:** Groq (Llama 3), Whisper API
- **Authentication:** Clerk
- **Other Tools:** TensorFlow.js, Framer Motion
